Detailed Description

A class for opening a snoop file in read-only mode. This class enable to open the file and read all packets, packet-by-packet

A constructor for this class that gets the snoop full path file name to open. Notice that after calling this constructor the file isn't opened yet, so reading packets will fail. For opening the file call open()
