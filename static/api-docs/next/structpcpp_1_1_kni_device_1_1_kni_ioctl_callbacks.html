Detailed Description

New callbacks for KNI device events. This structure MUST be used ONLY when KniDeviceList::callbackVersion returns KniDeviceList::KniCallbackVersion::CALLBACKS_NEW. Or if You are sure that DPDK version used is 17.11 or higher. If some callback is not provided (nullptr) the request will always succeeds if other is not specified in callback description.

Note
This callbacks are direct copy of one defined in rte_kni_ops. Future maintainers of KNI device feature MUST refer to rte_kni_ops structure in rte_kni.h header file of DPDK to track the difference in signatures
